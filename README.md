# Coroa Store

Base inicial do front-end do e-commerce B2B em `React + TypeScript + Vite`, organizada para evoluir por componentes.

## Estrutura

- `src/app`: casca principal da aplicacao
- `src/components/ui`: kit base de componentes reutilizaveis
- `src/features/home`: secoes da landing inicial e previews de fluxo
- `src/styles`: tema global e tokens visuais
- `.specify`: fluxo do Spec Kit para especificacao, plano e tarefas
- `.agents/skills`: skills instaladas para o fluxo `speckit` no Codex

## Como rodar

```bash
npm install
npm run dev
```

## Fluxo orientado por IA

Antes de desenvolver uma feature nova, siga o fluxo do Spec Kit:

1. `$speckit-constitution`
2. `$speckit-specify`
3. `$speckit-plan`
4. `$speckit-tasks`
5. `$speckit-implement`
